# Daniel Tavares - Professional Website

A clean, minimalist professional website showcasing my work experience, projects, and online presence.

## Features

- **Responsive Design**: Mobile-first approach with clean, elegant styling
- **Interactive Elements**: Random text effect and smooth animations
- **Timeline**: Professional work experience with visual timeline
- **Projects**: Showcase of personal projects with live demos and GitHub links
- **Social Links**: Direct links to GitHub, LinkedIn, X (Twitter), and Discord profiles

## Design Philosophy

Inspired by clean, minimalist websites like [karpathy.ai](https://karpathy.ai) and [danperks.dev](https://danperks.dev), this website focuses on:

- Clean typography using Inter font
- Subtle animations and hover effects
- Mobile-responsive design
- Fast loading with no external dependencies (except Google Fonts)
- Accessible and semantic HTML structure

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with Flexbox and Grid
- **Vanilla JavaScript**: Interactive elements and animations
- **No frameworks**: Pure web technologies for optimal performance

## Getting Started

Simply open `index.html` in a web browser or deploy to any static hosting service.

## Customization

To customize the content:

1. **Personal Information**: Update the header section in `index.html`
2. **Work Experience**: Modify the timeline section with your experience
3. **Projects**: Replace the project cards with your own projects
4. **Social Links**: Update the social media links in the connect section
5. **Random Text**: Modify the `professionalTitles` array in `script.js`

## Deployment

This website is designed to be hosted on GitHub Pages. Simply push the files to your `username.github.io` repository and GitHub Pages will automatically serve the website.
